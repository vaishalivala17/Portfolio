## Key Features

- **Three.js Particle Background** — Interactive 3D particle field with floating wireframe geometries and mouse parallax
- **Custom Cursor** — Animated custom cursor with trailing ring effect
- **Typing Effect** — Rotating text animation for role titles
- **Scroll Reveal** — Smooth fade-up animations triggered by intersection observer
- **3D Card Tilt** — Interactive tilt effect on skill and project cards
- **Responsive Design** — Fully responsive layout for mobile, tablet, and desktop
- **Smooth Scrolling** — Seamless anchor navigation
- **Glitch Lines & Corner Decorators** — Cyberpunk-inspired UI accents

## Getting Started

To run this portfolio locally:

```bash
# Clone the repository
git clone https://github.com/vaishalivala17/vaishali-portfolio.git

# Navigate into the folder
cd vaishali-portfolio

# Open in your browser
open index.html
```

Or simply serve with a local server:

```bash
npx serve .
```
